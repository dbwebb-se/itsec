Rapport mall till LaTeX
===============================

I katalogen ligger två exempel på enklare rapportmallar för LaTeX tillsammans med en exempelfil för hur man kan hantera referenser.

Ett populärt onlineverktyg för att skriva och samarbeta kring LaTeX-dokument är [Overleaf](https://www.overleaf.com/).

Här kan du prova ut [rapportmallen i Overleaf](https://www.overleaf.com/read/dgqqnggjzzry).
